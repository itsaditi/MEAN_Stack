To create node structure-
npm init backend

To install express-
npn install express -save

Create JS file in Backend folder called server.j

To run the node app- node server.js

To read json message, install body-parser.
npm install body-parser -save

To enable cors-
npm install cors

Install Mongodb driver- npm install mongodb

Install Mongoose- npm install mongoose -- save

In order to send and retrive messages from mongoose, we need schema and model.

To save encrypted password in Mongo- npm i bcrypt --save
